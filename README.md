# Simple Billing System for a Store

A beginner-friendly Python billing system project where users can add items, remove items, view bill, and view total amount.

## Features
- Add item name and price
- Remove items from the bill if they want
- View the current bill where users can see the currently added items and their prices
- View the total cost of all the items combined

## How to Run
1. Save both files: main.py and billing_manager.py
2. Run:
   python main.py
